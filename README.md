# ECE243-ASM-LAB-1
An introduction to assembly language programming using the nios v processor on the CPUlator simulator and the DE1-SoC hardware.

The goal of this lab was as follows:

- Write a Nios V Assembly language program that computes the sum of the numbers from 1 to 30, in a loop that explicitly computes the sum (i.e. don’t use a formula to compute the sum, use a loop).
- The sum must be placed into register s1 when the program is finished, and the program should go into an infinite loop when it is done, as shown in Figure 1 at the bottom. Test your program by compiling and running it on CPUlator.

The code is stored in part3.s and can be run in the online simulation software CPULator.
